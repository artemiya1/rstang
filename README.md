# RSTA NG - فورک V2RayNG با اتصال خودکار و دور زدن فیلترینگ

**RSTang** یک فورک اصلاح‌شده از [V2RayNG](https://github.com/2dust/v2rayNG) است که با قابلیت‌های مدیریت خودکار اتصال، اسپوف ترافیک و تکنیک‌های پیشرفته برای عبور از فیلترینگ تقویت شده است.

## ✨ ویژگی‌ها
- 🔄 **اتصال خودکار** – اتصال را پس از قطعی یا تغییر شبکه به‌صورت خودکار برقرار می‌کند.
- 🛡️ **اسپوف (Spoofing)** – الگوهای ترافیک را تغییر می‌دهد تا از تشخیص DPI (بازرسی عمیق بسته) جلوگیری کند.
- 🚀 **دور زدن فیلترینگ** – بهینه‌سازی شده برای محیط‌های با سانسور شدید.
- 📦 **سبک و بهینه** – بر پایه آخرین نسخه هسته V2Ray با مصرف باتری کم.
